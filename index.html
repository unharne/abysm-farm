> ПРОГРАММА [v]

> ДОСТУПНЫЕ КОМАНДЫ

Ваша программа выполняется циклически. Используйте новые команды для навигации.

  • move('forward'): Перемещает дрон на 1 клетку в текущем направлении.
  • turn('left'/'right'): Поворачивает дрон на 90 градусов.
  • action('plant'): Сажает семя на почве (если есть семена).
  • action('harvest'): Собирает урожай.
  • drone.x = 8 / drone.y = 6: Сброс позиции.
  • drone.direction = 0..3: Сброс направления (0:UP, 1:RIGHT, 2:DOWN, 3:LEFT).

НОВЫЕ ФУНКЦИИ (УСЛОВИЯ): Вы можете использовать эти функции в коде, чтобы дрон выполнял действия, только если они возможны. Например: action('harvest') if can('harvest') (простая логика, которую парсер поймет).

  • can('harvest'): Возвращает true, если под дроном созревший урожай.
  • can('plant'): Возвращает true, если под дроном пустая почва и есть семена.
